# 🤔ㅣGPle?
"**GPle이란?** GSM Place의 줄임말로, GSM 아이디어페스티벌을 위해 7기 학생들이 한 팀이 되어 만든 프로젝트입니다.   
이 앱은 GSM의 다양한 장소에서 촬영한 사진을 **손쉽게 공유**하고, 위치별로 모인 사진들을 통해  
**서로의 추억**을 되새기며 **즐거운 순간**들을 함께 나누는 경험을 제공합니다."
